8月14日学习笔记

8/14学习记录

CRF学习

  1. 概念：（Conditional Random Field）条件随机场模型是一种判别式模型，它在观测序列的基础上对目标序列进行建模，重点解决序列化标注问题。当前流行的实现工具：CRF++。
  2. 应用场景：句法分析，序列标注，数据分割 ，组块分析等自然语言处理任务。在中文分词，中文任命识别，歧义消除等领域表现良好。
  3. 处理流程：
    • 将分词语料的标注符号转化为用于命名实体序列标注的标记。
    • 确定特征模板，一般采用当前位置的前后n个位置的字及其标记表示，即以当前位置的前后n个位置范围内的子串及其标记作为观察窗口（一般n取2~3）
    • 得到相应的特征函数
    • 训练CRF模型的参数λ和u
  4. CRF++使用方式：分为训练过程和测试过程两步

  • 数据格式

训练数据格式：第一列是需要标注的字或词，最后一列是输出位标记tag，额外的特征例如词性可以加在中间列，所以训练集或者测试集都至少需要两列

  • 特征模板

  1. 模板类型:

    ​ CRF++中的模板构建有两类，一类是Unigram标注，一类是Bigram标注，Unigram是比较常用的特征模板，组成的模板数量也比较多。Bigram模板比较简单，一般是当前词和前面一个词的自动组合生成的Bigram特征集合。 Unigram/Bigram是指输出token的Unigram/Bigrams，而不是特征的。

    • Unigram类型 ：每一行%x[#,#]生成一个CRFs中的点(state)函数：f(s,o),其中s为t时刻的标签（output），o为t时刻的上下文。
    • Bigram类型：每一行%x[#,#]生成CRFs中的边(Edge)函数：f(s’,s,o),其中s’为t-1时刻的标签，也就是说，Bigram类型与Unigram类型大致相同，只是还要考虑到t-1时刻的标签，如果直线一个B的话，默认生成f(s’,s).

“%x[row, column]” 代表获得当前指向位置向上或向下偏移|row|行，并指向第column列的值。

若当前词为”网”，则每个特征模板对应的字如下：

U00:%x[-2,0]=====>人

U01:%x[-1,0]=====>民

U02:%x[0,0]=====>网

U03:%x[1,0]=====>1

U04:%x[2,0]=====>月

U05:%x[-2,0]/%x[-1,0]/%x[0,0]=====>人/民/网

U06:%x[-1,0]/%x[0,0]/%x[1,0]=====>民/网/1

U07:%x[0,0]/%x[1,0]/%x[2,0]=====>网/1/月

U08:%x[-1,0]/%x[0,0]=====>民/网

U09:%x[0,0]/%x[1,0]=====>网/1

​ 以上述数据和特征模板来说：第一个模板U00:%x[-2,0]，当前词假设为”网”，输出output假设仅有”B、E、M、S”四种，可以产生如下四个特征函数：

func1=if(output=B and feature=’U00:人‘ )　return 1 else return 0

func2=if(output=E and feature=’U00:人‘ )　return 1 else return 0

func2=if(output=M and feature=’U00:人‘ )　return 1 else return 0

func2=if(output=S and feature=’U00:人‘ )　return 1 else return 0

  • ​ 其中output是指当前词的预测标记，也就是”网”的预测标记，每个模板都会把所有可能的标记都列出一遍，然后通过训练确定每种标记的权重，合理的标记在训练样本中出现的次数越多，对应的权重就高，不合理的标记在训练样本中出现的少，对应的权重就少。
  • ​ 其中feature指的是当前此的-2位置处对应的词或者对应的词的特征，由于在这里没有其他特征所以用字本身做特征，有的数据会有词性，那feature就会是”人”这个字的词性而不是字本身了。简而言之，feature的作用就是确定模板所确定的当前词和临近词。

​ 在利用模板生成特征函数时会把所有可能的特征函数都列出来，由模型通过训练决定每个特征的重要程度。

​ 假设输出序列的集合大小为L，那么训练数据的每一行都会产生L个特征函数，假设输入序列长度为N，那么一个Unigram将会产生L*N个特征函数。

特征模板编写思路：将所有的特征都编写进去。

特征模板编写格式：

  • #开头的为注释
  • 行与行之间可以有空行
  • Unigram的特征前使用字母U，而Bigram的特征前使用字母B。后面的数字用于区分特征，当然这些数字不是一定要连续。
